# CTA Category: Encouraging Post Saves

**File Purpose:** Use when the post is value-dense — a framework, checklist, step-by-step guide, or stat-heavy content that the reader will want to reference again later. The CTA must make saving feel like the obvious, self-serving choice for the reader. Never beg for saves — frame it as doing them a favour.

---

1. **The Busy Reader Save**
Purpose: Acknowledges the reader's time pressure and gives them permission to come back.
Example: In the middle of something? Save this post — it'll be here when you're ready.
Template: In the middle of something? Save this post to read it again later.

2. **The Milestone Save**
Purpose: Turns saves into a collective goal with a reward attached — drives community action.
Example: Save this post — if we hit 50 saves we'll turn it into a free screensaver!
Template: Save this post and if it gets to 50 we'll make it a free screensaver!

3. **The Giveaway Save**
Purpose: Incentivises saves with a flash giveaway milestone — boosts reach and engagement simultaneously.
Example: Save this post — if we hit 100 saves we'll run a flash giveaway!
Template: Save this post and if we get to 100 we'll run a flash giveaway.

4. **The Reply Later Save**
Purpose: Keeps the conversation open and makes saving feel like a social act, not just archiving.
Example: Too busy to reply right now? Save this — we'll be here when you're back.
Template: If you're too busy, save this post to reply later, we'll be here!
